# Shekemas Speedrun Tracker

This tool is called Zarokh in honor to the boss you are killing again and again.

So Zarokh allows you to track the time you last to kill Zarokh.

## Running

## Using

This tool is extremly simple.

You can move the window wherever you want to see the clock.

Click Start button to start to track your times.
Click Floor button when finish every floor.
Click Pause button if you take a break, and do not forget to click Start again on return.

Click + button to see your records.
